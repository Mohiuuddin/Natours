# 🏔️ Natours — Advanced CSS & Sass Nature Tours

A cutting-edge, highly animated landing page for an adventurous nature tour company. This project serves as a showcase for advanced CSS architecture, modern design patterns, and complex UI interactions without relying on heavy JavaScript libraries.

**🔗 [Live Preview](https://mohiuuddin.github.io/Natours/)** 

---

## 🛠️ Technical Deep Dive

This project isn't just about looks; it’s about **scalable and maintainable code**. 

### 1. Advanced CSS Architecture
* **Sass 7-1 Pattern:** Organized the entire project into a modular folder structure (abstracts, components, layout, pages, etc.) for enterprise-grade maintainability.
* **BEM Methodology:** Implemented **Block Element Modifier** naming conventions to ensure clear, conflict-free styling.
* **Responsive Design:** Used "Desktop-First" and "Mobile-First" strategies with custom Sass mixins for flexible media queries.

### 2. High-Level UI Features
* **Custom Grid System:** Built a proprietary float-based grid system (before moving to Flexbox/Grid) to understand deep layout logic.
* **CSS Animations:** Implemented complex `@keyframes` for the hero section and "card flip" 3D effects.
* **Modern Techniques:** Utilized `clip-path`, `background-blend-mode`, and `shape-outside` for unique organic shapes.
* **Checkbox Hack:** Created a fully functional navigation menu and popup modals using only CSS (zero JavaScript).

---

## 🚀 Key Features

* **3D Rotating Cards:** Interactive tour cards that flip to reveal pricing and call-to-actions.
* **Video Backgrounds:** Seamlessly integrated background videos for an immersive user "Stories" section.
* **Custom Forms:** Stylized radio buttons and input fields that push the boundaries of standard browser defaults.
* **Hover Effects:** Advanced "Composition" images that move and scale dynamically.

---

## 🎨 Tech Stack

* **HTML5:** Semantic structure for maximum SEO and accessibility.
* **Sass (SCSS):** Advanced use of mixins, functions, and variables.
* **NPM Scripts:** Automated the compilation, prefixing, and minification of CSS files.

## ⚙️ Development

To compile the Sass and run the project:
1. Clone the repo: `git clone https://github.com/Mohiuuddin/Natours-Project.git`
2. Install dependencies: `npm install`
3. Run the compiler: `npm run watch:sass`
